# MobX

MobX는 리액트 상태 관리 라이브러리입니다.

주요 개념들

## Observable State (관찰 받고 있는 상태)
MobX를 사용하고 있는 앱의 상태를 관찰합니다.
해서, 앱에서 사용하고 있는 값, 객체 등이 바뀌면 MobX는 어떤 부분이 바뀌었는 지 알 수 있습니다.

## Computed Value (연산된 값)
연산된 값은 기존의 상태값과 다른 연산된 값에 기반하여 만들어질 수 있는 값입니다.
간단하게, 연산 할 때, 연산에 기반되는 값이 바뀔 때만 새로 연산하고, 바뀌지 않는다면 기존 값을 그냥 사용합니다.

## Reaction (반응)
ComputedValue랑 비슷하다는데 이게 뭔지 몰라소... ㅎㅎ
특정 값이 바뀔 때 특정작업을 하게 한다는데...


## Actions (액션)
액션은 상태의 변화를 일으키는 것을 말합니다.
