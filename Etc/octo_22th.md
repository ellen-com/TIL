## Repository (저장소)
### Remote Repository (원격 저장소)
원격 서버에 저장된 저장소로, 여러 사람이 함께 공유한다.
### Local Repository (개인 저장소)
우리가 직접 관리하는 저장소로, 내 PC에 저장되어 있다.

## 중요 용어
### comit
파일을 추가, 제거하거나 변경한 내용을 저장소에 저장하는 작업.

### push
파일을 추가, 제거하거나 변경한 내용을 <b>원격</b>저장소에 업로드하는 작업.

### pull
원격 저장소에 저장되어 있는 내용을 로컬 저장소로 끌어오는 작업.

### branch
브랜치 정말 사랑합니다... 브랜치가 있어서 편하게 이것저것 할 수 이쒀...
브랜치란 독립적으로 어떤 작업을 진행하기 위한 개념입니다.
필요에 의해 만들어지는 각각의 브랜치는 다른 브랜치의 영향을 받지 않기 때문에, 여러 작업을 동시에 진행할 수 있습니다.

### merge
나뉘어진 브랜치를 다시 하나의 브랜치로 합치는 과정.
이때 오류 엄청 발생하니까 조심혀... 나두 이때 똥싸부러써

### stash
커밋 하지 않은 내용을 잠시 저장할 수 있도록 하는 명령어
잠깐 킵해두고 다시 꺼낼 수 있어서 증말 좋아요

명령어들은... 내일 합시다...
